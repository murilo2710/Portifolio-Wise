# Portfólio Pessoal - Trainee Wise Systems

Este projeto foi desenvolvido para o processo de **Trainee Wise Systems**, com o objetivo de criar uma página web que funcione como meu portfólio profissional, destacando minhas habilidades, projetos e experiências como desenvolvedor.

A proposta é apresentar as informações de forma clara, visualmente agradável e responsiva, utilizando boas práticas de desenvolvimento e versionamento.

## 📋 Descrição do Projeto

A página foi projetada para:

- Apresentar minhas informações pessoais e profissionais
- Exibir habilidades técnicas de forma visual, com cards e indicadores
- Mostrar projetos com descrição, tecnologias e links para repositórios
- Disponibilizar formas de contato por formulário e redes sociais

A estrutura foi desenvolvida com HTML5 e estilizada com CSS3, com interações em JavaScript para navegação, carrossel de projetos e modal de detalhes.
O código é versionado com Git e mantido no GitHub.

## 🚀 Funcionalidades

### Seção de Apresentação Pessoal

Foto, nome, objetivo profissional e links diretos para GitHub, LinkedIn e e-mail.

### Habilidades Técnicas

Exibição das principais competências técnicas e comportamentais em formato visual, com paginação e barras de nível.

### Projetos Anteriores

Carrossel com projetos em destaque e modal com descrição detalhada, tecnologias utilizadas e acesso ao repositório.

### Formulário de Contato

Canal para envio de mensagens e seção com contatos rápidos para facilitar conexões profissionais.

### Responsividade Completa

Layout adaptável para desktop, tablet e mobile, mantendo boa leitura e usabilidade em diferentes tamanhos de tela.

## 🛠️ Tecnologias Utilizadas

- **HTML5** - estrutura do conteúdo
- **CSS3** - estilização e identidade visual
- **JavaScript (ES6+)** - interações e comportamentos dinâmicos
- **Git & GitHub** - versionamento e repositório remoto

## 📦 Requisitos Técnicos
